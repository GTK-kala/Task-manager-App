> A **modern task management web app** built with **React**, **Node.js**, **Express**, and **MySQL**.  
> It features **JWT authentication**, **project & task management**, and a **Kanban board** with drag & drop.

---

## ✨ Features

- 🔐 **JWT Authentication** – secure login & register  
- 📂 **Projects Management** – create and manage projects  
- ✅ **Tasks Management** – assign tasks, update status, set priority  
- 📊 **Kanban Board** – drag & drop tasks between columns  
- 👥 **User Roles Ready** – extendable for teams and permissions  
- ⚡ **RESTful API** – clean and well-structured endpoints
